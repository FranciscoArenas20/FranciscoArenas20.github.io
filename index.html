📱 InventAgro — Aplicación Móvil de Gestión de Productos

Aplicación móvil desarrollada como prueba técnica que consume una API con 50,000 registros simulados de productos agroindustriales. Implementa lista infinita con scroll eficiente (ListView.builder), búsqueda local sobre datos cacheados, y carga optimizada de imágenes sin re-descargas. El backend dockerizado en NestJS con PostgreSQL implementa paginación cursor-based, relaciones entre tablas y permisos. La arquitectura frontend sigue una separación limpia de capas (Data Sources, Repositories, Providers, UI) con inyección de dependencias mediante get_it y estrategia de caché con Riverpod + Hive para disponibilidad offline. Desplegado en Railway con APK generado para distribución.

Flutter Dart NestJS PostgreSQL Riverpod Hive Docker Railway Clean Architecture
